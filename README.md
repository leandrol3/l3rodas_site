# Atualize os arquivos no servidor FTP
Utilizar a extensão SFTP
Configurações 
``` {
    "name": "L3 Rodas Server",
    "host": "ftp.l3rodas.com.br",
    "protocol": "ftp",
    "port": 21,
    "username": "l3rodas",
    "password": "********",
    "remotePath": "httpdocs/",
    "uploadOnSave": true
} ```
Clicar com o botão direto e selecionar Upload
